# Check Design

Checklist rápida y ejemplos de uso para `check:tailwind` y `test:a11y`.

- Comprueba presencia de `tailwind.config.ts` y mapeo de tokens.
- Verifica `src/components/ui/README.md` existe si hay componentes nuevos.
- Ejecuta `npm run test:a11y` para pruebas manuales de accesibilidad.

Ejemplo de uso local:

```bash
npm run check:tailwind
npm run lint:tailwind
npm run test:a11y
```
