# Design System NeuralForge

Bienvenido al Design System modular para proyectos AI Studio. Aquí encontrarás todo lo necesario para crear interfaces consistentes, accesibles y escalables.

## ¿Qué incluye?

- **Design Tokens:**
  - Archivo [design-tokens.json](examples/design-tokens.json) con colores, radios y animaciones base.
  - Guía para mapear tokens en Tailwind y estilos globales.

- **Componentes Modulares:**
  - Ejemplo de [Button](components/Button.tsx) con variantes y buenas prácticas.
  - Patrón para crear nuevos componentes reutilizables.

- **Validación y Accesibilidad:**
  - Scripts para validar configuración de Tailwind ([scripts/check-tailwind-config.js](scripts/check-tailwind-config.js)).
  - Validación de accesibilidad con axe-core ([scripts/run-axe.js](scripts/run-axe.js)).
  - Checklist de calidad ([examples/check-design.md](examples/check-design.md)).

- **Integración y Ejemplos:**
  - Ejemplo de integración en [examples/integracion.md](examples/integracion.md).
  - Plantillas para autenticación con Supabase y Firebase.

- **Extensión del Sistema:**
  - Proceso documentado en [EXTENSION.md](EXTENSION.md) para añadir tokens, componentes y ejemplos.

## ¿Cómo empezar?

1. Revisa los design tokens y configura Tailwind.
2. Usa los componentes base y extiende según tus necesidades.
3. Ejecuta los scripts de validación antes de publicar.
4. Consulta los ejemplos y sigue el proceso de extensión para mantener la calidad.

---

> Este sistema garantiza coherencia visual, accesibilidad y escalabilidad en todos tus proyectos AI Studio.
