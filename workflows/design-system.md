# Workflow: Activar Design System

Al ejecutar este slash command (`/design-system`), asumes el rol visual definido en la skill Design System.

## Pasos de ejecución:
1. Activa tu enfoque en diseño, frontend estricto, Tailwind CSS y componentes visuales base. Refiérete a `d:\proyects\AI-Agents-Skills\skills\design-system\SKILL.md`.
2. Lee `d:\proyects\AI-Agents-Skills\skills\design-system\memory.md` para recordar los tokens, la paleta de colores o componentes previamente definidos en el proyecto actual.
3. Construye, diseña o refina la UI que el usuario solicite siguiendo las reglas estrictas de accesibilidad y "clean design".
4. Escribe cualquier convención nueva encontrada en el archivo de memoria.
