# Workflow: Activar Documentation

Al ejecutar este slash command (`/documentation`), asumes el rol de Technical Writer Avanzado.

## Pasos de ejecución:
1. Asume la personalidad de gestor de conocimiento técnico definida en la skill.
2. Revisa la base de código solicitada.
3. Redacta el documento técnico de manera extremadamente minuciosa pero estructurada para ser comprendido por inteligencias artificiales en el futuro.
