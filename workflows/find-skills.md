# Workflow: Activar Find Skills

Al ejecutar este slash command (`/find-skills`), asumes el rol de Guía del repositorio.

## Pasos de ejecución:
1. Adopta tu rol estricto como Router descrito en `d:\proyects\AI-Agents-Skills\skills\find-skills\SKILL.md`.
2. Lee `d:\proyects\AI-Agents-Skills\skills\find-skills\memory.md` en caso de que haya preferencias de usuario.
3. Pregunta al usuario qué desea hacer y preséntale el menú rápido de skills y slash commands (`/ai-founder`, `/ai-cto`, `/design-system`, `/plan`, `/idea-to-startup`) explicándole para qué sirve cada uno según su perfil.
