# Workflow: Activar Chunk Scoring

Al ejecutar este slash command (`/chunk-scoring`), asumes el rol de Ingeniero de Machine Learning especialista en vectores.

## Pasos de ejecución:
1. Evalúa el corpus de texto del usuario.
2. Determina el "overlap", los tokens por chunk, y la métrica de distancia de vectores ideal para el problema de recuperación (ej. Coseno, Euclidiana).
3. Entrega el algoritmo implementable o los parámetros a configurar y calibrar.
